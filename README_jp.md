## PhantomCipher (ファントムサイファー)

[English](./README.md) | [简体中文](./README_zh.md) | [日本語](./README_jp.md)

PhantomCipher は、Obsidian 向けにシームレスな透明暗号化体験を提供します。

> [!CAUTION]
> - 本プラグインは個人利用を目的としており、極めて重要な機密情報には推奨されません。
> - 重要なファイルを暗号化する前に、**必ず保管庫のバックアップを作成してください**。
> - データ紛失に関して、開発者は一切の責任を負いません。
> - **パスワードを紛失すると、ファイルは二度と復元できません。**

### ✨ 主な機能

- **Argon2id + AES-GCM**: 業界標準のアルゴリズムを採用し、高度なセキュリティを実現。
- **透明な暗号化**: 読み書きをインターセプトします。エディタでは通常通り編集でき、ディスク上では暗号化されて保存されます。
- **圧縮機能**: Deflate 圧縮を内蔵し、Base64 エンコードによるファイルサイズの肥大化を抑制します。
- **パフォーマンス**: 「セッションソルト」最適化により、大規模な保管庫でもスムーズな動作を実現。
- **安全な保存**: マスターパスワードはシステムのセキュアなキーチェーンに保存されます。

### 🚀 クイックスタート

1. プラグイン設定で **マスターパスワードの設定** を行います。
2. **リンク...** をクリックしてシステムキーチェーンを開き、**シークレットを追加...** をクリックします。ID（名前）を設定し、実際のパスワードを **シークレット値** に入力して保存・選択します。
3. **動作モード** を「自動暗号化」に変更します。
4. **リボンアイコン** または **右クリックメニュー** から、特定のファイルやフォルダの暗号化状態を手動で切り替えます。

## 🛠️ 復元ツール
Obsidian 以外でファイルを復号する必要がある場合（プラグインが利用できない、またはデータをエクスポートしたい場合など）、PhantomCipher CLI 復号ツールをご利用ください：
[PhantomCipher CLI 復号ツール](https://github.com/Lumingtianze/phantom-decrypt)
