# PhantomCipher (幻影加密)

[English](./README.md) | [简体中文](./README_zh.md) | [日本語](./README_jp.md)

PhantomCipher 为您的 Obsidian 库提供无缝的透明加密体验。

> [!CAUTION]
> - 本插件仅供个人使用，请勿用于处理高度机密或核心业务数据。
> - 在加密重要文件前，**请务必备份您的仓库**。
> - 作者不对任何形式的数据丢失负责。
> - **遗忘密码将导致文件永久无法找回。**

### ✨ 核心特性

- **Argon2id + AES-GCM**: 采用行业标准算法，提供极高的安全性。
- **透明加解密**: 拦截底层读写逻辑；你在编辑器中正常书写，文件在磁盘上以密文存储。
- **内置压缩**: 使用 Deflate 压缩算法，抵消 Base64 编码带来的体积膨胀。
- **极致性能**: 引入“会话盐”优化，确保在大规模库中依然拥有流畅的编辑体验。
- **安全存储**: 密码存储在系统的安全钥匙串中。

### 🚀 快速开始

1. 在插件设置中找到 **设置主密码** 选项。
2. 点击 **链接...** 打开系统钥匙串，点击 **添加密钥...**。为它起一个 ID（名称），并将你的实际密码填入 **密钥值** 中，保存并选中该条目。
3. 将 **运行模式** 设置为“自动加密”。
4. 通过 **功能区图标** 或 **右键菜单** 手动转换单个文件或文件夹的加密状态。
